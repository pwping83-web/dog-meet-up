# 05 — 가상 파트너 광고 예시 (한시 무료 → 월 유료)

> **데모·기획용 가상 카피**입니다. 실제 업체·수료·취업을 암시하지 않도록 운영 시 문구를 조정하세요.

## 5.1 배너 한 줄

**돌봄·맡기기 직업 훈련** — 지금은 **한시 무료 노출**, 이후 **월 광고**로 계속 노출

## 5.2 목록용 짧은 본문

반려견 **방문 돌봄·맡기기**를 직업으로 하고 싶은 분을 위한 기초·실무 안내(가상).  
**강아지 돌보는 일**과 **가르치는 일(강사)**이 늘어나 **일자리가 하나 더** 생기는 그림을 목표로 합니다.

**노출 정책(가상):** 처음에는 플랫폼에서 **한시적으로 무료**로 노출해 보고, 이후에는 **한 달 단위 유료 광고**로 전환될 수 있습니다.

## 5.3 면책 한 줄

본 내용은 **예시 광고**이며, 실제 교육·자격·취업을 보증하지 않습니다.

---

*실제 광고 상품을 판매할 때는 약관·청약철회·과장 금지·세금 처리를 별도로 두는 것이 좋습니다.*

# 03 — 광고·홍보·성장 (실행 순서)

> 법률·표시광고법·플랫폼 약관은 **직접 검토·준수**가 필요합니다. 아래는 **일반적인 실행 순서**입니다.

## 3.1 단계별 로드맵

| 단계 | 목표 | 예시 액션 |
|------|------|-----------|
| **0. 준비** | 말 한 가지로 설명 가능 | 한 줄 피치, 스크린샷 5장, 데모 URL |
| **1. 유기(무비용)** | 초기 사용자·피드백 | 지인 견주, 지역 반려동물 카페·산책 모임 공유 |
| **2. 콘텐츠** | 검색·공유 되게 | 블로그/숏폼: “동네에서 모이자 쓰는 법”, “맡기기 글 예시” |
| **3. 커뮤니티 파트너** | 신뢰 전이 | 훈련소·미용실·병원(동의 시) 안내 카드·QR |
| **4. 유료 실험** | 단가 학습 | 소액 메타/구글 지역 타깃, **한시 무료 노출 → 월 구독** 같은 파트너 광고 모델 문서화(`05` 참고) |
| **5. 리텐션** | 재방문 | 알림(PWA), 채팅·즐겨찾기, 돌봄 인증 콘텐츠 |

## 3.2 채널별 체크리스트

- **앱 내:** 홈 배너, 글쓰기 완료 후 공유 버튼, 고객센터·법적 고지 링크 명확히.
- **SNS:** 인스타·스레드·네이버 카페 — **과장 금지**, “데모/베타” 표기 권장.
- **SEO:** 타이틀·메타 설명에 지역+반려견+모임 키워드; 블로그 URL 구조 단순화.
- **오프라인:** 명함·스탠드에 QR + “앱 설치 아님, 웹 접속”(PWA 안내).

## 3.3 메시지 축 (카피 방향)

- **공감:** “산책 친구부터 맡기기까지, 동네에서 한번에.”
- **신뢰:** “인증 절차·노출 정책을 제품에 넣었습니다.” (실제 기능과 일치하게)
- **행동:** “지금 동네 글 둘러보기” / “돌봄 맡기기 쓰기”

## 3.4 지표 (최소)

- DAU/WAU, 글 작성 수, 가입 전환, `create-meetup` 시작→완료, 돌봄 탭 클릭.
- 유료 전환 시: 광고 노출 수, 클릭, 문의 전환.

## 3.5 파트너·교육 광고(수익 확장 아이디어)

- **한시 무료 노출**로 파트너를 모으고, **월 정액**으로 목록·배너를 고정하는 식의 **가상 시나리오**는 `05-virtual-partner-ad-sample.md` 참고.
- 실제 계약 시: 광고 심의·세금계산서·취소·환불 정책을 별도 약관으로 정리.

## 3.6 하지 말 것

- 허위 후기·비인증 돌봄을 인증처럼 표현.
- 채팅에 개인정보 과다 노출 유도.
- API 키·관리자 계정을 홍보 콘텐츠에 노출.
